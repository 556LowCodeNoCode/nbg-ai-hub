The llm 0.32a2 release introduces support for the new OpenAI /v1/responses endpoint, enabling interleaved reasoning across tool calls for GPT-5 class models. This update allows users to see summarized reasoning tokens during prompt runs, enhancing transparency and debugging for advanced LLM workflows.
